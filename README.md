# StayTuned Cordova SDK - Demo Ionic Application (Ionic Angular)

This repository is a demo application for the Staytuned Cordova SDK

## Requirements

-   Ionic-Cli > 6
-   npm > 6

## Install

```bash
yarn install
ionic build android
```

## Run

```
ionic run android
```
